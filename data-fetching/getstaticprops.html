With Incremental Static Regeneration

Add the revalidate property to enable ISR:

export async function getStaticProps() {
  const res = await fetch('https://api.example.com/posts')
  const posts = await res.json()

  return {
    props: {
      posts,
    },
    revalidate: 60, // Revalidate every 60 seconds
  }
}

Context Object

The getStaticProps function receives a context object with the following properties:

  • params - Route parameters for dynamic routes
  • preview - Preview mode information
  • previewData - Preview data
  • locale - Active locale (if enabled)
  • locales - All supported locales
