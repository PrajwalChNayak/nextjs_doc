getServerSideProps

getServerSideProps runs on every request and allows you to fetch data that changes frequently or is user-specific. 

// pages/dashboard.js
export default function Dashboard({ user, data }) {
  return (
    

      
Welcome {user.name}

      
{data.message}

    

  )
}

export async function getServerSideProps(context) {
  const { req, res, query } = context
  
  // You can use the request object to access cookies, headers, etc.
  const userRes = await fetch('https://api.example.com/user', {
    headers: {
      Cookie: req.headers.cookie,
    },
  })
  const user = await userRes.json()
  
  const dataRes = await fetch(`https://api.example.com/data?id=${query.id}`)
  const data = await dataRes.json()

  return {
    props: {
      user,
      data,
    },
  }
}

Context Object Properties

  • req - HTTP IncomingMessage object
  • res - HTTP ServerResponse object
  • params - Route parameters for dynamic routes
  • query - Query string object
  • preview - Preview mode information
  • previewData - Preview data
  • resolvedUrl - Normalized request URL
  • locale - Active locale (if enabled)
  • locales - All supported locales

Performance Note

Use getServerSideProps only when you need to pre-render a page whose data must be fetched at request time. Consider client-side fetching if the data doesn't need to be pre-rendered.
