📄 Pages in Next.js

Creating pages in Next.js is as simple as creating files! Each page is just a React component that Next.js automatically turns into a route. Let's see how easy it is! ✨

🎯 How It Works (Super Simple!)

📁

1. Create a File

Make a .js file in the pages folder

⚛️

2. Export Component

Write a simple React component and export it as default

🚀

3. Magic!

Next.js automatically creates a route for you!

👨‍💻 Let's Build Your First Page!

1

Create the file

💡

File Location Tip:

Create a file called about.js inside your pages folder

2

Write your component 

// pages/about.js
export default function About() {
  return (
    

      
About Us

      
Welcome to our amazing website!

      
We love building awesome things with Next.js! 🚀

    

  )
}
✅

That's it!

You've just created a page! Next.js will automatically make it available at http://localhost:3000/about

🎨 More Examples to Try

🏠 Home Page (index.js)

The index.js file becomes your home page at / 

// pages/index.js
export default function Home() {
  return (
    

      
Welcome to My Website! 🎉

      
This is the home page

    

  )
}

📍 Available at: http://localhost:3000/

📧 Contact Page 

// pages/contact.js
export default function Contact() {
  return (
    

      
Get in Touch! 📬

      
Email us at: hello@example.com

      
We'd love to hear from you!

    

  )
}

📍 Available at: http://localhost:3000/contact

🎯 Key Points to Remember

📝

File = Route

Every file in pages/ becomes a route automatically

🏠

index.js = Home

pages/index.js becomes your homepage

⚛️

React Components

Each page is just a React component

🚀

No Config Needed

Next.js handles routing automatically!
