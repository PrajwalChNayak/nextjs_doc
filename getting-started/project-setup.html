Project Setup

After creating your project, navigate to the directory and start the development server: 

cd my-app
npm run dev

Open http://localhost:3000 with your browser to see the result.

Project Structure

Here's what your Next.js project structure looks like:

my-app/
├── pages/
│   ├── api/
│   ├── _app.js
│   └── index.js
├── public/
├── styles/
├── next.config.js
├── package.json
└── README.md

Key Directories

  • pages/ - File-system based router. Each file becomes a route
  • public/ - Static assets like images, fonts, etc.
  • styles/ - CSS files for styling your application
  • pages/api/ - API routes for backend functionality

Hot Reloading

You can edit pages/index.js and see the changes update in real-time thanks to Fast Refresh.

Development Commands

Development

npm run dev

Start development server

Production Build

npm run build

Start Production

npm run start

Start production server
