🛠️ Setting Up Your Project

Great! You've created your Next.js project. Now let's get it running and explore what you've got! This is where the magic begins! ✨

1

📁 Navigate to Your Project

First, let's enter your project folder. Think of this as opening the door to your new app! 

cd my-awesome-app
💡

Remember:

Replace "my-awesome-app" with whatever you named your project!

2

🚀 Start Your Development Server

Now for the exciting part! Let's bring your app to life. This command starts a local server so you can see your app in the browser. 

npm run dev
⏱️

What's happening?

Next.js is starting up your development server. This might take a few seconds the first time!

3

🌐 See Your App Live!

The moment you've been waiting for! Open your browser and navigate to your local development server.

🎯 Open this URL in your browser:

http://localhost:3000

You should see your Next.js welcome page! 🎉

📂 Understanding Your Project Structure

Let's explore what Next.js created for you! Think of this as a tour of your new digital home. 🏠 

my-awesome-app/
├── pages/           👈 Your website pages live here!
│   ├── api/         👈 Backend API endpoints
│   ├── _app.js      👈 App wrapper (affects all pages)
│   └── index.js     👈 Your homepage (what you see at /)
├── public/          👈 Images, icons, and static files
├── styles/          👈 CSS files for styling
├── next.config.js   👈 Next.js configuration
├── package.json     👈 Project info and dependencies
└── README.md        👈 Project documentation
📄

pages/

The Heart of Your App

This is where the magic happens! Each file automatically becomes a page on your website.

Example: pages/about.js becomes yoursite.com/about

🖼️

public/

Your Static Assets

Put your images, icons, and other static files here. They'll be available at the root of your site.

Example: public/logo.png becomes /logo.png

🎨

styles/

Make It Beautiful

Your CSS files live here. This is where you'll make your app look amazing!

Tip: Start with globals.css for site-wide styles

⚡

pages/api/

Backend Power

Create backend endpoints here! Perfect for handling forms, databases, and more.

Example: api/hello.js becomes /api/hello

🔥 Hot Reloading Magic!

Here's the best part: when you edit your code, you'll see changes instantly in your browser! No need to refresh or restart anything.

Try it now: Open pages/index.js in your code editor, make a change, and watch your browser update automatically! ✨

🛠️ Essential Commands

Here are the key commands you'll use during development. Don't worry, you only need a few!

🚀

Development Server

For coding and testing

npm run dev

Start your development server with hot reloading. Use this while building your app!

📦

Production Build

Ready for deployment

npm run build

Creates an optimized version of your app for production. Super fast and efficient!

🌟

Start Production

Run the built app

npm run start

Runs your production build. Use this after running npm run build.

🔍

Code Linting

Check for issues

npm run lint

Checks your code for potential problems and style issues. Keeps your code clean!
