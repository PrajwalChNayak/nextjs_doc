Installation

To create a new Next.js application, you can use create-next-app which sets up everything automatically for you.

System Requirements

  • Node.js 16.8 or later
  • macOS, Windows (including WSL), and Linux are supported

Automatic Installation

We recommend creating a new Next.js app using create-next-app, which sets up everything automatically for you.

npx create-next-app@latest my-app

TypeScript Projects

To create a TypeScript project, use the --typescript flag:

npx create-next-app@latest my-app --typescript

Manual Installation

Install next, react and react-dom in your project:

npm install next react react-dom

Add the following scripts to your package.json:

{
  "scripts": {
    "dev": "next dev",
    "build": "next build",
    "start": "next start",
    "lint": "next lint"
  }
}
