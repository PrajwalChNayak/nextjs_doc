üîß Manual Setup (For Advanced Users)

Want more control? Set up Next.js manually! This is perfect if you're adding Next.js to an existing project or want to understand how everything works under the hood.

1 Install the required packages

Install Next.js, React, and React DOM in your project:

npm install next react react-dom

2 Add scripts to your package.json

These scripts let you run your app in development, build it for production, and more:

{
  "scripts": {
    "dev": "next dev",
    "build": "next build", 
    "start": "next start",
    "lint": "next lint"
  }
}
