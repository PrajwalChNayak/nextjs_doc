Your First App

Let's create your first page. Next.js uses a file-system based router built on the concept of pages.

Creating a Page

Create a file called about.js in the pages directory:

// pages/about.js
export default function About() {
  return (
    

      
About Us

      
This is the about page

    

  )
}

Now you can access this page at http://localhost:3000/about.

Adding Styles

Let's add some CSS to make our page look better. Create a CSS module:

/* styles/About.module.css */
.container {
  max-width: 800px;
  margin: 0 auto;
  padding: 2rem;
}

.title {
  color: #0070f3;
  font-size: 2rem;
  margin-bottom: 1rem;
}

.description {
  font-size: 1.2rem;
  line-height: 1.6;
  color: #666;
}

Now update your about page to use these styles:

// pages/about.js
import styles from '../styles/About.module.css'

export default function About() {
  return (
    

      
About Us

      

        Welcome to our Next.js application! This is a simple example
        of how to create pages and add styling in Next.js.
      

    

  )
}
