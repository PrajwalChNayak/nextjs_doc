Step 1: Creating Your First Page

Let's create an "About" page for your website

Quick tip: Create a file named about.js in the pages folder

pages/about.js 
// pages/about.js
export default function About() {
  return (
    <div>
      <h1>About Us</h1>
      <p>This is the about page</p>
    </div>
  )
}

ðŸŽ‰ That's it!

Visit http://localhost:3000/about to see your new page!

Step 2: Making It Look Amazing

Add CSS styles to make your page beautiful

CSS Modules: Create reusable styles that won't conflict with other pages

styles/About.module.css 
/* styles/About.module.css */
.container {
  max-width: 800px;
  margin: 0 auto;
  padding: 2rem;
}

.title {
  color: #0070f3;
  font-size: 2rem;
  margin-bottom: 1rem;
}

.description {
  font-size: 1.2rem;
  line-height: 1.6;
  color: #666;
}

Now let's use these styles in our page. Update your about.js file:

pages/about.js (updated) 
// pages/about.js
import styles from '../styles/About.module.css'

export default function About() {
  return (
    <div className={styles.container}>
      <h1 className={styles.title}>About Us</h1>
      <p className={styles.description}>
        Welcome to our Next.js application! This is a simple example
        of how to create pages and add styling in Next.js.
      </p>
    </div>
  )
}

Step 3: Connecting Your Pages

Add navigation so users can move between pages easily

Link Component: Next.js provides a special Link component for fast page navigation

pages/index.js (home page) 
// pages/index.js
import Link from 'next/link'
import styles from '../styles/Home.module.css'

export default function Home() {
  return (
    <div className={styles.container}>
      <main className={styles.main}>
        <h1 className={styles.title}>
          Welcome to <a href="https://nextjs.org">Next.js!</a>
        </h1>
        
        <nav className={styles.nav}>
          <Link href="/about">
            <a className={styles.card}>Go to About Page â†’</a>
          </Link>
        </nav>
      </main>
    </div>
  )
