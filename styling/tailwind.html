Tailwind CSS

Tailwind CSS is a utility-first CSS framework that provides low-level utility classes to build custom designs.

Installation

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

Configuration

Update your tailwind.config.js file:

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    './pages/**/*.{js,ts,jsx,tsx}',
    './components/**/*.{js,ts,jsx,tsx}',
  ],
  theme: {
    extend: {
      colors: {
        primary: {
          50: '#eff6ff',
          500: '#3b82f6',
          900: '#1e3a8a',
        }
      }
    },
  },
  plugins: [],
}

Add Tailwind Directives

Add the Tailwind directives to your CSS file (e.g., styles/globals.css):

@tailwind base;
@tailwind components;
@tailwind utilities;

Benefits of Tailwind CSS

  • Utility-first: Build complex components from simple utility classes
  • Responsive design: Built-in responsive modifiers
  • Dark mode support: Easy dark mode implementation
  • Customizable: Extend the framework with your own utilities
  • Small bundle size: Only includes the CSS you actually use
