How to Add Tailwind to Next.js

Get up and running in 3 simple steps

1

Install Tailwind CSS

Add Tailwind and its dependencies to your Next.js project: 

npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p

âœ¨ What this does

This installs Tailwind CSS and creates a basic configuration file. The -p flag also creates a PostCSS config file!

2

Configure Tailwind

Update your tailwind.config.js file to include your Next.js paths: 

/** @type {import('tailwindcss').Config} */
module.exports = {
  content: [
    './pages/**/*.{js,ts,jsx,tsx}',
    './components/**/*.{js,ts,jsx,tsx}',
    './app/**/*.{js,ts,jsx,tsx}',
  ],
  theme: {
    extend: {
      colors: {
        primary: {
          50: '#eff6ff',
          500: '#3b82f6',
          900: '#1e3a8a',
        }
      }
    },
  },
  plugins: [],
}
3

Add Tailwind Directives

Add the Tailwind directives to your global CSS file (e.g., styles/globals.css): 

@tailwind base;
@tailwind components;
@tailwind utilities;

ðŸŽ‰ You're ready to go!

That's it! Now you can use Tailwind classes anywhere in your Next.js components.
