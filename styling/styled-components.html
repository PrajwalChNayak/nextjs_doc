Styled Components

Styled Components is a CSS-in-JS library that allows you to write CSS directly in your JavaScript components.

Installation

npm install styled-components
npm install --save-dev babel-plugin-styled-components

Configuration

Add the styled-components plugin to your .babelrc file:

{
  "presets": ["next/babel"],
  "plugins": [["styled-components", { "ssr": true }]]
}

Basic Usage

import styled from 'styled-components'

const Container = styled.div`
  max-width: 1200px;
  margin: 0 auto;
  padding: 2rem;
`

const Title = styled.h1`
  font-size: 2.5rem;
  color: ${props => props.primary ? '#007bff' : '#333'};
  margin-bottom: 1rem;
`

const Button = styled.button`
  background: ${props => props.primary ? '#007bff' : 'transparent'};
  color: ${props => props.primary ? 'white' : '#007bff'};
  border: 2px solid #007bff;
  padding: 0.5rem 1rem;
  border-radius: 4px;
  cursor: pointer;
  
  &:hover {
    background: #0056b3;
    color: white;
  }
`

export default function Home() {
  return (
    
      Welcome to Next.js!
      
      
    
  )
}

Theming

import { ThemeProvider } from 'styled-components'

  colors: {
    primary: '#007bff',
    secondary: '#6c757d',
    success: '#28a745',
    danger: '#dc3545',
  },
  breakpoints: {
    mobile: '768px',
    tablet: '1024px',
  }
}
