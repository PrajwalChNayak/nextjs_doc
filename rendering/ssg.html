Static Site Generation (SSG)

Static Site Generation is a method where pages are generated at build time. This results in fast loading times and better performance.

Benefits of SSG

  • • Pages are generated on-demand
  • • Automatic static optimization
  • • No rebuild required for new pages

Static Generation without Data

By default, Next.js pre-renders pages using Static Generation without fetching data: 

function About() {
  return 
About

}

export default About

Static Generation with Data

Use getStaticProps to fetch data at build time:

export async function getStaticProps() {
  const res = await fetch('https://api.example.com/posts')
  const posts = await res.json()

  return {
    props: {
      posts,
    },
  }
}
