Server-Side Rendering (SSR)

Server-side rendering allows you to pre-render pages on each request. This is useful for pages that need to show frequently updated data. 

export async function getServerSideProps(context) {
  const res = await fetch(`https://api.example.com/data`)
  const data = await res.json()

  return {
    props: { data }
  }
}

Benefits of SSR

  • Pages are always up-to-date
  • Better for SEO with dynamic content
  • Access to request context (cookies, headers)

Performance Note

SSR can be slower than Static Generation since the server must compute the result on every request.
