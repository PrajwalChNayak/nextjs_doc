Key Features

  • Automatic Format Optimization: Serves modern formats like WebP and AVIF
  • Responsive Images: Automatically generates multiple sizes
  • Lazy Loading: Images load only when they enter the viewport
  • Placeholder Support: Built-in blur placeholders
  • Priority Loading: Disable lazy loading for above-the-fold images

Configuration

Configure image domains in next.config.js for remote images:

/** @type {import('next').NextConfig} */
const nextConfig = {
  images: {
    domains: ['example.com', 'another-domain.com'],
  },
}

module.exports = nextConfig
