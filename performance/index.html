Built-in Optimizations

  • Automatic Code Splitting: Pages are automatically split into separate bundles
  • Prefetching: Link components automatically prefetch pages
  • Image Optimization: Next.js Image component optimizes images automatically
  • Font Optimization: Automatically optimize and load Google Fonts
  • Script Optimization: Optimize loading of third-party scripts

Performance by Default

Next.js optimizes your application automatically, but understanding these optimizations helps you make better decisions.
